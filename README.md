# Gushwork Assignment Implementation

This repository contains a pixel-perfect implementation of the home page specified in the Figma design.

## Features
- **Sticky Header**: Appears when scrolling down past the first fold and scrolling back up. It uses a modern glassmorphism aesthetic with a blur effect.
- **Interactive Carousel**: A responsive product carousel showcasing premium appliances.
- **Image Zoom**: Hovering over carousel images triggers a detailed zoom view with a magnifying lens effect.
- **Fully Responsive**: Optimized for Desktop, Tablet, and Mobile devices.
- **Clean Codebase**: Uses vanilla HTML5, CSS3, and modern JavaScript.

## Files
- `index.html`: Main structure with semantic tags.
- `styles.css`: Styling with a modern dark theme, Flexbox/Grid systems, and custom animations.
- `script.js`: Logic for the sticky header, carousel transitions, and image zoom tracking.
- `assets/`: Directory containing generated high-quality product images.
